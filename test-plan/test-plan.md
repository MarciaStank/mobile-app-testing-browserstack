# 📘 Test Plan – Shopping List Mobile App

## 🎯 Objective
Validate the functionality and input behavior of the Shopping List Android application, focusing on user input handling and item creation flow.

---

## 📌 Scope
The following features are covered in this test plan:

- New Item field  
- Quantity field  
- Item creation flow  

---

## 🧪 Test Types
The following testing approaches were applied:

- Functional Testing  
- Exploratory Testing  
- Input Validation  
- Boundary Testing  

---

## ⚙️ Test Environment
Tests were executed under the following conditions:

- Platform: Android  
- OS Version: Android 11  
- Device: Samsung Galaxy S21  
- Execution: Real device cloud using BrowserStack  

---

## 🚫 Out of Scope
The following areas were not covered:

- Backend validation  
- Performance testing  
- Security testing  

---

## ⚠️ Risks
Potential risks identified during testing:

- Unknown or undefined input validation rules  
- Lack of documented requirements
